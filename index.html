Lenguaje HTML

Versión actual: HTML 5

HTML es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera.

El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, trabajador del CERN propuso un nuevo sistema de "hipertexto" para compartir documentos.

Una de las características de HTML es la forma en que se divide la información. Esto es por medio de dos grandes bloques: el elemento HTML o head y el cuerpo o contenido.
